This Jupyter Notebook contains the full workflow for analyzing crime patterns in Travis County, Texas, 
from 2016 to 2023. It includes data cleaning, feature engineering, and logistic regression modeling to 
explore the relationship between reported crime hotspots and socioeconomic indicators such as race, poverty, 
education, and renter occupancy. The notebook also includes visualizations and spatial mapping using Python 
libraries like GeoPandas, Seaborn, and Matplotlib. Outputs include correlation heatmaps, odds ratio summaries, 
and choropleth maps of predicted hotspot probabilities.
